# FreeRoulette

Inspired by my 3 visits to casinos in Sydney, a roulette webapp that wouldn't make me broke (or rich)

## Features

- Standard roulette table (European and American styles)
- Mobile‑friendly responsive design (mostly)

## Planned Features

- Spin logs and charts (result analysis, streaks, win/loss tracking)
- Probability calculator (show odds of different bets)
- Mass user‑rule betting (simulate strategies)
- Dark/light mode toggle (maybe not)
- Custom/fun tables (customizable wheel, variants like 1/3 tiles as zero) (this feels too complicated after leanring how the numbers are placed around the standard wheels)
- Custom themes/skins for table and wheel
- Export data (CSV/JSON for deeper analysis)
- AI dealer commentary (realtime LLM commentary on playstyle, win/loss) (might be hard to implement on Github Pages)

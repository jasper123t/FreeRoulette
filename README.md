# FreeRoulette

A full-stack roulette portfolio project starter.

## Structure
- `frontend/`: Bun + Vite + React frontend
- `backend/`: Go ServeMux API backend
- `shared/`: shared constants and utilities
- `docs/`: documentation
- `tests/`: test plan and information

## Quick start
1. Install dependencies in the frontend:
   - `cd frontend`
   - `bun install`
2. Start the frontend:
   - `bun run dev`
3. Start the backend:
   - `cd backend`
   - `go run src/main.go`
